# Rapport de stage – ICUBE IMIS – TC-FOOT

## Auteur
William PALOMBIER

## Contexte
Ce dépôt contient mon rapport de stage de Master Intelligence des Données de Santé, réalisé au laboratoire ICube, équipe IMIS, dans le cadre du projet TC-FOOT.

## Sujet
Étude morphoanatomique par segmentation du Cavum Vergae dans une cohorte de footballeurs professionnels à partir d’IRM 3D MPRAGE.

## Contenu du dépôt
- `rapport_stage_william_palombier.pdf` : rapport complet
- `figures/` : illustrations éventuelles

## Résumé
Ce travail porte sur la segmentation manuelle du Cavum Vergae sur IRM, la quantification volumétrique, l’analyse descriptive des résultats et les perspectives d’automatisation par IA.

## Mots-clés
IRM, segmentation, Cavum Vergae, TC-FOOT, imagerie médicale, intelligence artificielle
