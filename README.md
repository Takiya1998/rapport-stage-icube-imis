# rapport-stage-icube-imis
Projet TC-FOOT au laboratoire ICube (IMIS, Strasbourg) : segmentation manuelle du cavum vergae sur IRM 3D T1 (MPRAGE) afin d’analyser sa prévalence, sa morphologie et son volume chez d’anciens footballeurs vs témoins. Objectif : standardiser l’évaluation et poser les bases d’une future automatisation par deep learning.
